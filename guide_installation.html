Installer le tableau de bord
Les Enfants de Pondypatch

Ajoutez le tableau de bord sur votre ecran d'accueil ou votre bureau — il s'ouvrira comme une application, sans taper l'adresse.

Ouvrir le tableau de bord
Adresse https://sites.google.com/view/les-enfants-de-pondypatch/accueil

Smartphone & Tablette

Android — Chrome
  1. Ouvrez Chrome et allez sur le tableau de bord
  2. Appuyez sur les 3 points en haut a droite
  3. Selectionnez "Ajouter a l'ecran d'accueil"
  4. Nommez le raccourci et validez
iPhone / iPad — Safari
  1. Ouvrez Safari et allez sur le tableau de bord
  2. Appuyez sur l'icone Partage (carre avec une fleche vers le haut)
  3. Faites defiler et choisissez "Sur l'ecran d'accueil"
  4. Appuyez sur "Ajouter" en haut a droite

Ordinateur

Windows — Chrome ou Edge
  1. Ouvrez Chrome ou Edge et allez sur le tableau de bord
  2. Cliquez sur les 3 points en haut a droite
  3. Chrome : Plus d'outils → Creer un raccourci, cochez "Ouvrir dans une fenetre"
  4. Edge : Applications → Installer ce site en tant qu'application
Mac — Safari ou Chrome
  1. Ouvrez Safari ou Chrome et allez sur le tableau de bord
  2. Safari : dans la barre de menus, Fichier → Ajouter au Dock
  3. Chrome : meme methode que Windows (le raccourci ira dans le Launchpad)
Une fois installe, cliquez simplement sur l'icone Les Enfants de Pondypatch sur votre ecran d'accueil ou bureau pour ouvrir directement le tableau de bord — sans navigateur, sans taper l'adresse.
